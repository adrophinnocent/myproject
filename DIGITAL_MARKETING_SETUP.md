# Digital Marketing Setup for Twina Safaris

This guide outlines the essential configurations needed to turn this website into a high-conversion lead generation engine.

## 1. Analytics & Tracking
*   **Google Analytics 4 (GA4):** Ensure the `google_analytics_id` is set in the Admin Panel to track visitor behavior.
*   **Facebook Pixel:** Essential for retargeting travelers who viewed a tour but didn't book.

## 2. Lead Generation
*   **Newsletter:** All subscribers are stored in the `newsletter_subscribers` table. Use these for weekly safari deals.
*   **WhatsApp Marketing:** The "WhatsApp Enquiry" buttons use the `site_whatsapp` setting. Ensure this remains `255795482197`.

## 3. Review Management
*   **In-house Reviews:** Encourage guests to use the "Write a Review" feature on tour pages.
*   **Moderation:** Admin must manually approve reviews via the Admin Dashboard before they appear live.
