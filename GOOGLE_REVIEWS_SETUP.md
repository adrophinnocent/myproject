# Google Reviews Integration Setup

Integrating Google Reviews builds massive trust for international tourists.

## 1. Google Places API
*   Go to Google Cloud Console.
*   Enable **Places API**.
*   Generate an **API Key**.

## 2. Place ID
*   Find the **Place ID** for your Moshi office.
*   Add the Key and ID to your website Settings.

## 3. Benefits
The website will automatically fetch and display your 5-star ratings from Google Maps directly on the homepage.
