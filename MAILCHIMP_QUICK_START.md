# Mailchimp Quick Start

## Fast Setup (3 Steps):
1.  **API Key:** Paste your key in `.env`.
2.  **Verify Domain:** Ensure `twinasafaris.com` is verified in Mailchimp so your emails don't go to spam.
3.  **Test:** Sign up for the newsletter on the website footer and check if you appear in Mailchimp.

**Note:** Always use the high-end luxury email templates to match the website branding.
