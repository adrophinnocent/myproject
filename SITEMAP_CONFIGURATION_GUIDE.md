# Sitemap Configuration Guide

A sitemap tells Google which pages exist on Twina Safaris.

## 1. Automatic Generation
The sitemap is dynamically generated at `twinasafaris.com/sitemap.xml`. It automatically includes:
*   All Published Tours.
*   All Published Blog Posts.
*   Main pages (Home, About, Contact).

## 2. Google Search Console
1.  Log in to Search Console.
2.  Go to **Sitemaps**.
3.  Enter `sitemap.xml` and click **Submit**.
