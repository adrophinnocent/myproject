# Social Auth Setup (Login with Google/Facebook)

Allowing travelers to login with their social accounts increases trust and bookings.

## 1. Google Auth
*   Create a project in **Google Cloud Console**.
*   Add `GOOGLE_CLIENT_ID` and `GOOGLE_CLIENT_SECRET` to `.env`.

## 2. Facebook Auth
*   Register at **Meta for Developers**.
*   Configure the "Login with Facebook" product.
*   Add credentials to `.env`.
